Python
How to use:

1. Download Python 3.6 or above and Pycharm.
2. Run the setup file of Pycharm and configure the interpreter.
3. Make sure folders like mr_nodes, mr_gpi, pulseq2jemris and sequence are all in same folder.
4. Open pycharm and add interpreter path.
5. Go to File, click on setting, then project interpreter in that settings, then opt more, click on show path for selected interpreters (last option in the list) and 
   add the path for the folder mr_gpi, mr_nodes and pulseq2jemris and click on apply.
4. Load the sequence and execute
5. Seq file will be generate and waveform will be displayed for 1 TR
