MATLAB
How to use:

1. Should have MATLAB R2015a or above.
2. Download pulseq which is a open source framewrok for development, representation and execution of MR.
3. Sequences and mr+ folder should be in same folder.
4. Go to Code sequences in that tested sequences and choose any sequence and execute in MATLAB.
5. It will ask the path name, choose MATLAB folder.
6. Seq file will be generated and waveform for 1 TR will be displayed.
